CONVERSOR DRIVE → URL DIRECTA

Fátima Pro · Thumbnail / vista directa

Pega tus enlaces de Google Drive (uno por línea, o el ID suelto). El conversor extrae el ID y genera la URL directa tipo thumbnail que el sistema usa para imágenes — lista para pegar en la Biblioteca, tarjetas del hub o cualquier bloque.

Resultados

Pega tus enlaces arriba y pulsa “Convertir”.
¿Cómo obtener el enlace en Drive?
Clic derecho en el archivo → Compartir → Copiar enlace. Importante: cambia el acceso a Cualquier persona con el enlace para que las imágenes se vean en el sistema público.
El conversor acepta cualquier formato de Drive (/file/d/ID/view, open?id=ID, thumbnail?id=ID) e incluso el ID solo.
